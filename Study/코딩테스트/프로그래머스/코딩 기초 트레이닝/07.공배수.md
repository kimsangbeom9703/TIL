
# [프로그래머스] 코딩 기초 트레이닝

## 07 -  공배수


### 질문
```
정수 number와 n, m이 주어집니다. number가 n의 배수이면서 m의 배수이면 1을 아니라면 0을 return하도록 solution 함수를 완성해주세요.
```

### 답변
```java
class Solution {
    public int solution(int number, int n, int m) {
        return number % n == 0 ? ( number % m == 0 ? 1 : 0) : 0;
    }
}
```
