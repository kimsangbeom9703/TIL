
# [프로그래머스] 코딩 기초 트레이닝

## 05 -  원소들의 곱과 합


### 질문
```
정수가 담긴 리스트 num_list가 주어질 때, 모든 원소들의 곱이 모든 원소들의 합의 제곱보다 작으면 1을 크면 0을 return하도록 solution 함수를 완성해주세요.
```

### 답변
```java
class Solution {
    public int solution(int[] num_list) {
        int a = 1;
        int b = 0;
        for(int i=0; i<num_list.length;i++){
            a *= num_list[i];
            b += num_list[i];
        }
        return a > (b*b) ? 0 : 1;
    }
}
```
