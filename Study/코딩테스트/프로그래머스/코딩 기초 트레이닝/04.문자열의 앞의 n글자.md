
# [프로그래머스] 코딩 기초 트레이닝

## 04 -  문자열의 앞의 n글자


### 질문
```
문자열 my_string과 정수 n이 매개변수로 주어질 때, my_string의 앞의 n글자로 이루어진 문자열을 return 하는 solution 함수를 작성해 주세요.
```

### 답변
```java
class Solution {
    public String solution(String my_string, int n) {
        String answer = my_string.substring(0,n);
        return answer;
    }
}
```
