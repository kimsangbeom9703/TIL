# [JAVA] #1 Java 시작하기 

## 자바란?
```
    자바는 객채 지향 프로그래밍 언어이고 초기에는 가전제품에 탑재해 동작하는 프로그램을 만들기 위해 탄생했으나 지금은 
    웹 , 모바일 앱 개발에 가장 많이 사용한다.
    현재 우리나라 기업에서 사용하는 프로그램의 80%는 자바로 만들어져있다.
```

## 자바로 할 수 있는 일
```
    1. 웹 프로그래밍
    2. 안드로이드 애플리케이션
    3. 게임 개발
    4. 데이터베이스 처리
    5. 빅 데이터 및 분산 처리
```

## 자바로 할 수 없는 일
```
    1. 시스템 프로그래밍
    2. 높은 성능을 요구하는 프로젝트
    3. ios 애플리케이션 개발
```

## JDK 설치하기
```
    개발환경을 구축해 보자
    자바 프로그래밍의 필수 도구인 JDK(java development kit) 설치
    jdk 는 코드작성,소스를 컴파일하는 컴파일러 등으로 이루어져 있다

    1. https://www.oracle.com/java/technologies/downloads/

```
