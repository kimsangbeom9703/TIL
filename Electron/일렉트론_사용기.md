# [Electron] 일렉트론 사용기

##  일렉트론 사용이유

```
    웹으로 앱을 만들 수 있기 때문에 선택하였다.
    앱 ex) 쉽게 말하면 exe 로 설치되는 파일이라고 보면 된다. 

    내가 주로 작업한 것은 웹뷰에 내가 만든 사이트를 보여주고 
    그 중간 영역에 다른 웹페이지를 띄우고 싶은 경우 사용했다.

    이번 작업 내용은 프로그램 실행 시 인증키 를 입력하고 sqlite3에 저장을 하고
    값이 있을 경우 ? 인증된 사이트를 보여주고 없는경우 계속 인증키를 입력하는 페이지를 보여주려고 한다.        
```
