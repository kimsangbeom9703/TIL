# [Electron] 일렉트론 퀵스타트

## 참고
<https://github.com/electron/electron-quick-start>

##  일렉트론 퀵스타트
```
    git clone https://github.com/electron/electron-quick-start
    cd electron-quick-start
    npm install
    npm start

    끝..
```

##  폴더 구조
```bash
    electron-quick-start/
        node_modules/
        .gitignore
        index.html
        LICENSE.md
        main.js
        package.json
        package-lock.json
        preload.js
        README.md
        renderer.js
        styles.css
```
