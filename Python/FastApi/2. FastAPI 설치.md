##  가상 환경
```
    1.  cd virtualenv // 가상환경 폴더 이동
    2.  virtualenv server_fastapi // 가상환경 
        안될경우 / python -m venv server_fastapi
    3.  source server_fastapi/bin/activate
```

##  FastAPI 설치
```
    * 가상 환경에서 실행
    pip install fastapi
    python -m pip install --upgrade pip // pip update
```
