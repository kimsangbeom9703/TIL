# [FastApi] 폴더구조 
```
    문서    :   <https://phsun102.tistory.com/63>

    django 나 ci 처럼 프로젝트 폴더 구조가 있고 사용자 특성에 맞춰 변경해서 사용하는 거라고 생각했는데
    fastApi는 구조가 따로 존재하지않았다.(베이직 구조가 있다면 찾아서 다시 올리겠습니다.)
```

## 폴더 구조.
```
    예제에 올린 구조를 보면 app ,frontend 폴더가 있다
    app 폴더는 backend 즉 fastapi 가 동작하는 폴더이고
    frontend 폴더는 react가 작성될 폴더이다.
```

### app tree
```
app:.
├─apis
│  └─api.py
├─core
│  └─config.py
├─crud
│  └─crud.py
├─db
│  ├─models
│  │  └─model.py
│  └─connection.py
│  └─session.py
├─routes
│  └─router.py
└─main.py

api     :   mvc패턴의 컨트롤러 (db의 데이터를 받아 가공한다)
core    :   환경설정 정보 제공 .env파일 
curd    :   orm 형식의 db쿼리문 저장 폴더
db      :   db 연결 , 모델 정의
routes  :   url 로 api 나눈다.
main.py :   fastapi 실행 및 메인
```
