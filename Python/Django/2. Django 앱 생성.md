## 프로젝트와 앱의 차이
```
    프로젝트는 웹 사이트에 대한 구성 및 앱의 모음이고 앱은 하나의 앱이 여러개의 프로젝트에 포함 될 수 있다.
    컴포넌트 라고 생각하면 편하다.
```

## 앱 설치하기
```
    python manage.py startapp 앱이름
```
### 폴더 구조
```
앱이름/
    __init__.py
    admin.py
    apps.py
    migrations/
        __init__.py
    models.py
    tests.py
    views.py
```
