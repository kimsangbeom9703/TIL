# [TypeScript] 타입스크립트 설치하기

## 설치하기
```bash
# 글로벌 설치
# npm install --global typescript
$ npm i -g typescript

# 로컬 설치
# npm install --save-dev typescript
$ npm i -D typescript
```
```
두개의 차이는 전역으로 설치하냐 인데 모든 프로젝트에 사용하려면 글로벌로 설치하고
해당 프로젝트에만 설치하려면 로컬 설치를 진행하자.
```
```bash
# nodeJs에서 바로 사용하려면 ts-node를 설치한다.
$ npm i -D ts-node
# ts-node index.ts
```
